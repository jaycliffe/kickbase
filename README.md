# KickBase 2.0 — Club MIS

A modern Football Club Management Information System built as a single-page web app.

## Features
- 🔐 User authentication (login / register)
- 📊 Dashboard with live stats
- 👥 Squad & player management
- 📅 Fixtures & results tracker
- 💰 Club finances (income / expense)
- 🏆 League standings
- ⚙️ Club profile & settings
